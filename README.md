# ODIN-Etch-a-Sketch
\
TOP Etch a Sketch project specification overview.\
\
The webpage will have an nxn grid array which will respond when the mouse moves in and out of individual squares.  The user will see a javascript prompt when clicking the button.\
\
The webpage will:
- provide the user a button which will allow for the number of squares to specified (not to exceed 100).
- when clicking the button the existing grig will be removed and replaced by the new one.  Colors and 
  opacity will be reset.
- the grid will live in a space on the webpage with fixed pixel size.  Grid squares will need to be
  scaled to fit in the space provied.
- Squares will be assigned a random color as the mouse passes over them,
- Individual squares are darkened by 10% each time the mouse passes over it.  After 10 passes the square
  will be black.
